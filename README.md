# Sleep Tracker Telegram Bot

A Telegram bot that tracks sleep using simple buttons.

## Features

🌙 Bed — record when you go to sleep  
☀️ Wake — record when you wake up  
📊 Stats — view sleep statistics for 7 and 30 days  

The bot calculates:

- sleep duration
- sleep score
- sleep comments

## Tech

Python  
aiogram  
SQLite  

## Setup

1. Install dependencies  
pip install -r requirements.txt

2. Create a .env file  
BOT_TOKEN=your_token_here

3. Run the bot  
python bot.py
